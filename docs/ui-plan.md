# UI Plan

## Slide 1～4 の役割

### Slide 1

- 役割: 現在値と最近の変化要約
- 表示:
  - 現在値
  - 最近の変化
  - 雨の手がかりの要約
  - 今の並びと雨が近い形の比較

### Slide 2

- 役割: 雨の手がかりの解釈
- 表示:
  - 気圧 / 湿度 / 明るさ の signal rows
  - 雨の手がかりの数
  - 今の並びと雨っぽい並びの比較
  - `雨が近い？` の問い

### Slide 3

- 役割: 短期傾向
- 時間窓: 15 分
- 表示:
  - 気圧 / 湿度 / 照度 の短期グラフ
  - `今、何が変わっている？` の問い

### Slide 4

- 役割: 長期傾向
- 時間窓: 120 分
- 表示:
  - 気圧 / 湿度 / 照度 の長期グラフ
  - `流れは続いている？` の問い

### Status

- 通常ループ外の補助画面
- footer の `STATUS / BACK` で出入り
- 通常の教材スライドより優先度は低い

## 表示優先順位

1. 現在値が読めること
2. 直近変化の方向が分かること
3. 短期と長期の違いが見えること
4. 雨の手がかりを比較できること
5. Status は補助的に見られればよいこと

## 制約

- 日本語対応は同一リポジトリ内で進める
- `config.h` の UI 言語設定で英語 / 日本語を切り替える
- 日本語は
  1. 文言テーブル化
  2. 日本語文言追加
  3. 日本語フォント適用
  4. 日本語レイアウト調整
  の順で進める
- 夜間は `Light` を Rain signs から外す
  - 条件:
    - 日没後または日の出前
    - 低 lux が一定時間継続
- `LUX RATE` は UI 主役から外し、表示を簡素化
- `Status` の操作性改善は優先度低め
